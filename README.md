# AlphaDesk — Trading Analysis & Review Platform

## Deploy su Vercel (gratis, ~10 minuti)

### 1. Vai su vercel.com → "Add New Project"
### 2. Collega il tuo GitHub e carica questa cartella
### 3. Framework: Next.js (auto-rilevato)
### 4. Clicca Deploy → in 2 minuti il sito è live

URL risultante: https://alphadesk.vercel.app (o simile)

## Moduli
- Dashboard ✅ — KPI, VIX, indici, AI insight
- Analisi Mercati ✅ — Indici globali, settori, calendario
